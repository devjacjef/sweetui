# sweetui

This is a library to support my own learning of **Vue.js**, **Typescript**, **Node package management** and **Vite**. The design is simple, composable, data-driven components built from the bottom-up to quickly create frontends for my personal projects. This project is built in tandam with [Beehive](https://github.com/devjacjef/beehive).

# Acknowledments

This project is built by me however, **AI tools have been used** to provide guidance and review my code rather than build the whole library for me. This is meant to be a learning exercise that grows into a usable library.

# Installation for local projects

>Whilst I develop this library, I need to reflect my changes quickly. Both `npm` and `pnpm` provide the ability to do this with the `link` command. This documents how to use it.

First, install the dependencies for `sweetui` and then run the link command.

```sh
# assuming already inside sweetui
pnpm install
pnpm link
```

Afterwards, navigate to the project you want to install it for 

```sh
pnpm link sweetui
```
